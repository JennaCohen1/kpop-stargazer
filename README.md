# K-Pop Star Chart (Star Chart - K-Pop Demon Hunter)

Weekly interactive star chart for kids, K-Pop Demon Hunter themed.

- **Live app**: [https://kpop-star-chart.vercel.app/](https://kpop-star-chart.vercel.app/)
- **Code**: This GitHub repo
- **Data**: Stored locally

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Run locally

```sh
git clone <YOUR_GIT_URL>
cd kpop-stargazer
npm i
npm run dev
```

App runs at [http://localhost:8080](http://localhost:8080).

## Build

```sh
npm run build
```

## Deploy

The app is deployed on [Vercel](https://vercel.com). Connect this repo to a Vercel project; builds run on push. No Lovable or Supabase required.

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run preview` — preview production build locally
- `npm run lint` — run ESLint
- `npm run test` — run Vitest
